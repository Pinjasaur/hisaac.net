Open Source Projects

Tiime

In 2018, I set a goal for myself: To release an iOS app in Apple’s App Store. Tiime is the realization of that goal. I wanted to learn more about iOS development by creating an app from scratch, and to experiment with different iOS technologies and methodologies during the process.

  • Used the Coordinator pattern for the app’s architecture
  • Created without using any third party dependencies
  • Gained a deeper understanding of the view controller life cycle, UIKit, and building UIs in code
  • Built a somewhat non-standard API using primarily table views, navigation controllers, and a master split view controller
  • Added theme customization using UIAppearance and other UIKit methods
  • Had some fun implementing timers and date math to calculate the different clocks

Check out Tiime on GitHub for more information and to look at the code, or download the app from the App Store to see it for yourself.

Plain Pasta

Plain Pasta is a small macOS menu bar utility for keeping your clipboard clean. My goal with this project was to learn the basics of Swift development on macOS, and to help fill a need in my own workflow.

The app’s purpose is to strip any formatting from text on the clipboard, essentially making “Paste and Match Style” the default behavior on the macOS. It was a great experience learning about developing for macOS, and learning how to utilize some of the different APIs and methodologies it has to offer.

Check out Plain Pasta on GitHub for more information and to look at the code, or download the app to see it for yourself. Who knows, maybe it will find a place within your workflow as well.
